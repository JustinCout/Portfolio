Thanos' Survivors

A web app that brings together a Superhero API and the Reddit API in one cohesive concept. This application was composed with JavaScript and jQuery, using API calls to manipulate the JSON data of various superheroes and subreddits. The concept of the application is prefaced after the events of Avengers: Infinity War, creating a fun and interactive application for fans.

Tic-Tac-Toe

For my digital design final, I was tasked with creating a game and showcasing CSS animations. I used JavaScript and jQuery to create the game logic that determines turns, winning combinations, and draws. I then styled the game board with HTML and CSS, adding animations where they fit.

Critical Debates

Critical Debates is a simple chat application that is currently one-way. The purpose of this project was to create a chat application that uses XML to store chatlogs and AJAX to refresh the chats on an interval. I used PHP to create a login system and interact with XML (XML DOM). This application does not have a database, as XML is used to store logins and chat data. You can login through Google, or use the credentials "jc" and "password" to test the app.
